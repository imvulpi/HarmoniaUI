Detailed Description

Base node for any harmonia tests that are being performed in the editor using nodes.

Creating a custom test

This class is very open for new tests. To create a custom tests, simply inherit from it. Call PassTest or FailTest to pass or fail tests, These will invoke OnTestFinished which other classes can listen to, those methods are virtual as well (you can override them).

For restarting a test, just call Restart, if you need to include your own logic, you can do that by overriding the restart method.

This class directly doesn't introduce any tests, and shouldn't really be used.
