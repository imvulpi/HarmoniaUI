Checks whether SizingType.Border works as expected.

SizingType.Border should shrink the content size by padding and border sizes.

Most people want to use this option, due to not needing to recalculate or worry much about overflowing in changes to padding, or border sizes. It can shrink the content, so use it carefully.

Checks whether SizingType.Content works as expected.

SizingType.Content should enlarge the size set in styles by padding and border (or really anything other than content).

The advantage is that content size of nodes always remain the same, but the downside is that designers need to calculate sizes and spacings carefully to not overflow, size the content size can't shrink

Checks whether SizingType.Padding works as expected.

SizingType.Padding should enlarge the size set in styles by border. but shrink the content by padding.

It's an intermediate sizing between SizingType.Border and SizingType.Content
