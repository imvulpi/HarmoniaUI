Detailed Description

Node for testing the colors in a defined rectangle.

Tests whether all pixels in a rectangle are set to a specified ExpectedColor. Sizing can break at pixel values thus, TolerancePx allows setting a tolerance. it shrinks the checking rectangle by the value set (in Pixels).
