Detailed Description

Represents the parsed style for harmonia nodes after processing raw style data.

ParsedStyle is created by interpreting and validating the values from a Raw.StyleResource, converting them into strongly typed properties that can be easily computed to pixel values

Parsing does not convert values to pixels, nor does it apply inheritance, defaults, or cascading rules - those are handled later when producing the Computed.ComputedStyle.

Parsing is done to improve performance in recomputations, since there are different Units, parsing keeps it clean and fast to check and iterate over, when converting to pixels.
