Detailed Description

Responsible for converting a StyleResource into a ParsedStyle that can be used by the layout and visual systems.

The StyleParser reads the raw style data stored in a StyleResource, validates it, and converts it into strongly typed values stored in a ParsedStyle instance.

Parsing does not apply inheritance, or convert values to pixels - these are handled later when producing a Computed.ComputedStyle.
