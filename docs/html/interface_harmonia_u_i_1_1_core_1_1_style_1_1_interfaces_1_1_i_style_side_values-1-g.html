Detailed Description

Specifies the values of a style that describe rectangle sides Top, Bottom, Left, Right.

Depending on the style stage the T might be different:

  • Raw style -> strings
  • Parsed style -> typed structure (Usually
    See also
    Types.StyleSides<T>
    )
  • Computed style -> pixel values sides
